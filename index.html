Minimal Coding

Very minimal coding is required to set it up. You can be up and running within minutes of installation and even without setting up options prettyValidate is ready to go right out of the box.

HTML5 + CSS3

prettyValidate uses the latest practices in HTML5 so that you can be sure that your form will stay HTML5 compliant, but incase you're not using HTML5 prettyValidate will still work the same.

Configurable

prettyValidate is customizable in any way you can think of. From how the form is submitted, to what happens before or after the fields are validated, to callbacks from any errors it encounters.

Usage

Requirements

prettyValidate requires the use of jQuery v2.2 and up as well as Bootstrap v3.3 and up. You can get each of prettyValidates dependancies by following the links below:

  • jQuery v2.2.x (latest)
  • Bootstrap v3.3.x (latest)
  • colourBrightness.js (if you use the colorpicker)

If you would like to contribute your ideas to make prettyValidate more compatible with older versions of its dependancies, feel free to contact me in regards to becoming a contributor.
